# Dashboard横向布局调整计划

## 1. 移除居中容器
- 移除 `Main (2 columns)` 中的 `max-w-7xl.mx-auto` 居中容器
- 保留 `px-4` 作为页面最外层的左右padding
- 确保Dashboard使用全宽布局，内容更贴边

## 2. 调整主体网格布局
- 保持 `grid-cols-1.lg:grid-cols-[320px_1fr]` 的列宽比例
- 确保整体布局靠左，不要居中
- 调整gap值，保持适当的间距

## 3. 统一卡片内边距
- 检查所有卡片的padding设置
- 确保卡片内部padding控制在12~16px（p-3到p-4）
- 移除或调整任何使用p-6/p-8的卡片

## 4. 优化整体视觉效果
- 减少左右留白，提升信息密度
- 确保内容更"贴边"，接近真实Amazon Dashboard
- 保持卡片之间的适当间距，确保视觉层次感

## 5. 保持功能完整性
- 所有调整仅涉及布局和样式，不改变任何功能
- 确保所有组件正常工作
- 保持响应式设计，适配不同屏幕尺寸
