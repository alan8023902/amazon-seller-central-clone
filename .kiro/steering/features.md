# 功能模块文档

## 核心功能概览

这是一个高保真度的Amazon Seller Central克隆应用，包含完整的商家后台管理功能。

## 1. 认证系统 (AuthPages.tsx)

### 多步骤登录流程
- **邮箱输入页面** (`/auth/login-email`) - 输入邮箱地址
- **密码验证页面** (`/auth/login-password`) - 密码验证，支持"记住我"功能
- **两步验证页面** (`/auth/login-otp`) - OTP验证码输入
- **注册页面** (`/auth/register`) - 新用户注册

### 特性
- 表单验证使用 React Hook Form + Zod
- 多语言支持 (中文/英文)
- 像素级还原Amazon UI设计
- 会话状态持久化

## 2. 仪表板 (Dashboard.tsx)

### 主要组件
- **欢迎横幅** - 显示账户健康状态
- **全局快照** - 6列数据展示：销售、订单、消息、特色报价、反馈、支付
- **商品表现表格** - 显示商品详情、状态、销售数据
- **左侧操作卡片** - 待办事项和沟通模块

### 数据展示
- 实时销售数据和趋势图表
- 库存状态和商品管理
- 订单处理状态
- 多站点货币自动切换

## 3. 库存管理 (Inventory.tsx)

### 功能特性
- **商品列表** - 支持搜索、筛选、分页
- **状态管理** - Active/Inactive/Out of Stock状态切换
- **批量操作** - 选择多个商品进行批量编辑
- **商品详情** - SKU、ASIN、价格、库存数量显示
- **媒体预览** - 商品图片展示

### 交互功能
- 实时搜索 (商品名称、SKU、ASIN)
- 状态筛选器
- 响应式表格设计
- 加载状态和空状态处理

## 4. 业务报告 (BusinessReports.tsx)

### 报告类型
- **销售仪表板** - 销售趋势和关键指标
- **销售快照** - 订单项目、销售额、平均值统计
- **对比销售** - 图表和表格视图切换
- **业务绩效洞察** - 智能分析和建议

### 图表功能
- 使用 Recharts 库
- 支持时间序列数据对比
- 交互式图表工具提示
- 可切换图表/表格视图

## 5. 布局系统

### MainLayout
- **顶部导航栏** - 搜索、站点切换、语言切换、设置
- **二级菜单栏** - 主要功能模块导航
- **侧边抽屉** - 移动端友好的导航菜单
- **响应式设计** - 适配不同屏幕尺寸

### WithSidebarLayout
- **动态侧边栏** - 根据路径自动显示相应侧边栏
- **嵌套路由支持** - 支持多级页面结构
- **配置化导航** - 通过配置文件管理侧边栏结构

## 6. 国际化系统

### 多语言支持
- **语言切换** - 英文/中文实时切换
- **站点本地化** - 支持美国、日本、英国、德国、欧洲站点
- **货币自动转换** - 根据站点自动显示对应货币符号
- **文本翻译** - 完整的UI文本翻译

### 实现方式
- 使用 `useI18n` Hook
- 翻译文件结构化管理
- 参数化翻译支持

## 7. 状态管理

### Zustand Store
- **会话管理** - 用户登录状态、站点、语言设置
- **仪表板数据** - 销售数据、库存、订单信息
- **数据持久化** - LocalStorage自动保存
- **站点数据同步** - 切换站点时自动更新数据

### 数据流
- 全局状态集中管理
- 组件级状态使用React Hooks
- TanStack Query用于服务端状态管理

## 8. UI组件系统

### 核心组件
- **Card** - 统一的卡片容器
- **Button** - 多变体按钮 (黄色/白色)
- **InputField** - 表单输入组件
- **Logo组件** - 品牌标识

### 设计系统
- Amazon品牌色彩规范
- 统一的间距和字体规范
- 响应式设计原则
- 无障碍访问支持

## 9. 路由架构

### 路由结构
```
/auth/*                 # 认证相关页面
/app/dashboard          # 主仪表板
/app/inventory/*        # 库存管理 (带侧边栏)
/app/orders/*           # 订单管理 (带侧边栏)
/app/business-reports/* # 业务报告 (带侧边栏)
/app/ads/*              # 广告管理 (带侧边栏)
/app/settings/*         # 账户设置 (带侧边栏)
```

### 路由守卫
- 认证状态检查
- 自动重定向到登录页
- 路由状态保持

## 10. 数据模拟系统

### Mock数据
- **销售数据生成** - 基于站点的动态数据生成
- **商品数据** - 真实的商品图片和信息
- **图表数据** - 时间序列数据模拟
- **CSV/JSON导入** - 支持测试数据导入

### API模拟
- 异步数据加载模拟
- 错误状态处理
- 加载状态管理
- 站点特定数据转换

## 技术特色

1. **像素级还原** - 高度还原Amazon Seller Central的UI设计
2. **响应式设计** - 支持桌面和移动端
3. **性能优化** - 代码分割、懒加载、状态优化
4. **类型安全** - 完整的TypeScript类型定义
5. **现代化架构** - 使用最新的React 18特性
6. **开发体验** - 热重载、错误边界、开发工具集成
