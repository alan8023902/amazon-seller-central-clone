# Product Overview

This is a high-fidelity clone of Amazon Seller Central (US Market) built for educational and portfolio demonstration purposes. The application replicates the complex multi-step authentication flow, marketplace switching, localized dashboards, and data import/export capabilities of the real Amazon Seller Central platform.

## Key Features

- **Multi-step Authentication**: Pixel-perfect replication of Sign-In, Password, and Two-Step Verification flows
- **Marketplace Logic**: Support for multiple marketplaces (US, Japan, UK, Germany, Europe) with currency switching and localized data
- **Internationalization**: Full i18n support for English and Chinese languages
- **Data Management**: Import/export capabilities for test data with JSON/CSV support
- **Responsive Design**: Information-dense layout mimicking professional merchant dashboards
- **State Persistence**: Local storage persistence for user sessions and dashboard data

## Target Audience

This is a demonstration project showcasing modern React development practices and complex UI replication skills. It is not affiliated with Amazon and should not be used for actual e-commerce operations.
